# 🌟 Glowing Hover Effect Buttons

A collection of vibrant, neon, and glowing button hover effects built using **pure HTML5 and CSS3**. Designed to add a futuristic, eye-catching touch to modern web applications and landing pages.

---

## 🚀 Live Demo

Check out the interactive preview: **https://MALIK25252007.github.io/Glow-button/**

---

## 🛠️ Features

*   **Pure HTML & CSS:** No JavaScript or external libraries required.
*   **Neon & Ambient Glows:** Utilizes layered `box-shadow`, `drop-shadow`, and CSS pseudo-elements (`::before`, `::after`) for smooth lighting effects.
*   **Performance Optimized:** Smooth 60fps animations using hardware-accelerated CSS properties (`opacity`, `transform`).
*   **Theme Ready:** Built using CSS variables for easy color customization.
